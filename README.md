## Deploy on GitHub Pages
1. Create a new repo (e.g., `fintech-portfolio`)
2. Upload:
   - `index.html`
   - `assets/` folder
3. Repo → **Settings → Pages**
   - Source: Deploy from a branch
   - Branch: `main` / `(root)`
4. Open your live site.

## Notes
- Keep image filenames exactly as-is inside `assets/` to avoid broken images.
- Click any project image to open it in full size.
